# Unofficial Cthulhu Bot

This bot emulates dice rolling. It is tuned for CoC, but you can use this for general purposes.

**Note that  Chaosium Inc. owns the copyright of Call of Cthulhu.**

## Commands available

|Command|Frequently used|Description|
|:---|:---:|:---|
|`/선택`||주어진 항목 중 하나를 무작위로 선택합니다.|
|`/시트생성`|:star:|탐사자 시트를 생성합니다.|
|`/대항판정`||크툴루의 부름 7판 룰에 따라 대항 판정을 합니다.|
|`/다이스`|:star:|지정된 주사위를 굴립니다. Tyche 표현식을 지원합니다.|
|`/판정`|:star:|크툴루의 부름 7판 룰에 따라 기능 판정을 합니다.|

### Roll dices

Command: `/다이스` 주사위:`3d5 + 2d4`

### Attempts a skill roll.

Command: `/판정` 수치:`50` 기능이름:`자료조사`

### Create a character sheet

Command: `/시트생성` 이름:`탐사자` 근력:`60` 건강:`60` 크기:`65` 민첩:`70` 외모:`50` 지능:`65` 정신:`50` 교육:`75` 운:`50`
